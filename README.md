# Formulario-DB-PHP
<p>Formulário de confirmação de presença, criado em PHP, SQL, HTML, CSS e JavaScript, usando phpMyAdmin como Banco de Dados</p>
<h3>Acesse: https://unparliamentary-cha.000webhostapp.com/cadastros/FormularioPhpWebhost/index.php</h3>

</div>
<br><br>
<hr>
<p>As páginas dos usuários contém a opção do usuário consultar a descrição de diversos eventos disponíveis, trazidos do banco de dados, bem como confirmar sua presença para o evento</p>

<p>Acessando como "<i>gestor</i>" e om a senha "<i>123</i>" você terá acesso a página de Gestão onde é possível visualizar, criar e cancelar eventos, bem como verificar a confirmação de presença para cada evento</p>
